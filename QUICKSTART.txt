================================================================================
  FISCAL PRESSURE INDEX
================================================================================

Composite score (0-100) combining:
  • Debt-to-GDP ratio (higher = more pressure)
  • Interest as % of revenue (higher = more stress)
  • Annual deficit relative to GDP (higher = unsustainable)
  • Unemployment rate (higher = economic stress)
  • Fed funds + Treasury yields (higher = cost of borrowing)

Regime Thresholds:
  0-40   = STABLE      Green  ✓
  40-60  = ELEVATED    Yellow ⚠
  60-80  = CRITICAL    Orange ⚠⚠
  80-100 = CRISIS      Red    ✗

Current (Apr 4, 2026): ~58/100 (ELEVATED)
  Reasons: High debt-to-GDP, rising rates, unemployment ticking up

================================================================================
  TROUBLESHOOTING
================================================================================

"API won't start"
  → Check Python version: python3 --version (must be 3.9+)
  → Reinstall dependencies: pip install -r requirements.txt
  → Check port: lsof -i :8500

"No data showing"
  → Verify FRED_KEY: grep FRED_KEY .env
  → Test Treasury API: curl https://api.fiscaldata.treasury.gov/...
  → Force collection: python data_collector.py

"WebSocket not updating"
  → Check browser console for errors
  → Verify port 8500 accessible
  → Try full page refresh

"Database locked"
  → Check processes: ps aux | grep python
  → Backup & reset: cp debt_clock.db bak && rm debt_clock.db

See README.md and DEPLOYMENT.md for more troubleshooting.
